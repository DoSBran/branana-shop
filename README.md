# Next.js Tasks app
Para correr localmente, se requiere la base de datos
```
docker-compose up -d
```
# MongoDB url local
```
mongodb://localhost:27017/shopdb
```
# CrearDatos de prueba en la BD
Usar API:
```
http://localhost:3001/api/seed
```

# Instalar dependencias
Ejecutar:
```
yarn install
```
# Correr la app de nextjs localmente
Ejecutar:
```
yarn dev
```
